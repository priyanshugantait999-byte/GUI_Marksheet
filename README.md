📊 GUI Marksheet Generator (Python)
📌 Description

This project is a simple GUI-based marksheet application built using Python’s Tkinter library. It allows users to enter student details and grades, then automatically calculates total credits and SGPA.

🚀 Features
User-friendly graphical interface
Input fields for Name, Registration Number, and Roll Number
Grade-based credit calculation
Automatic Total Credit and SGPA calculation
Instant result display on button click
🛠️ Technologies Used
Python
Tkinter (GUI Library)
📚 Subjects & Credits
CS 201 → 4 credits
CS 202 → 4 credits
MA 201 → 3 credits
EC 201 → 4 credits
📐 Grading System
Grade	Points
A	10
B	9
C	8
D	7
P	6
F	0
⚙️ How It Works
User enters grades for each subject
System converts grades into credit points
Total credits are calculated
SGPA = Total Credits / 15
▶️ How to Run
Install Python (if not already installed)
Save the file as gui_marksheet.py

Run the program:

python gui_marksheet.py
📷 Output
Displays calculated subject credits
Shows total credit and SGPA in the GUI
📌 Note
The marksheet can be screenshotted and used as a report card
Built using basic Tkinter grid layout for simplicity
